## 🎯 Propósito del Proyecto

Selvadentro es una **landing page premium** para un proyecto de wellness real estate en Tulum, México. Inspirada en los principios de las **Blue Zones** y en el diseño de los mejores resorts wellness del mundo.

### Objetivo Principal
Convertir visitantes en leads calificados a través de una experiencia inmersiva que venda un **estilo de vida**, no solo propiedades.

### Referentes Analizados
- **One&Only Resorts** - Elegancia atemporal
- **Six Senses** - Experiencia sensorial
- **Chenot** - Ciencia del bienestar
- **Lanserhof** - Arquitectura médica

## 🎨 Sistema de Diseño

### Paleta de Colores
```
Primarios:
- Verde Selva Profundo: #1F3D2B
- Verde Selva Claro: #5A7D63
- Turquesa Cenote: #0FA3B1

Secundarios:
- Arena Natural: #E8DCC8
- Piedra Natural: #C9C4B8
- Blanco Marfil: #F7F5EF
```

### Tipografía
```
Headlines: Playfair Display (serif)
Body: Inter (sans-serif)
Micro: Inter uppercase tracking-wide
```

## 🚀 Stack Tecnológico Recomendado

```
Framework: Next.js 14 + TypeScript
Styling: Tailwind CSS
Animations: Framer Motion + GSAP
Icons: Lucide React
Fonts: Google Fonts
Images: Cloudinary / Next/Image
Video: HTML5 / Vimeo
Forms: React Hook Form + Zod
Analytics: Google Analytics 4
```
